# Solar Overview Card

[![hacs_badge](https://img.shields.io/badge/HACS-Custom-orange.svg)](https://github.com/hacs/integration)
[![GitHub Release](https://img.shields.io/github/release/YOUR_GITHUB_USER/solar-overview-card.svg)](https://github.com/YOUR_GITHUB_USER/solar-overview-card/releases)
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](LICENSE)

An animated Lovelace card for Home Assistant that visualises your solar energy system in real time — power flow diagram, live statistics for each source/sink, sparkline history charts and a scrollable device power list.

![Screenshot](screenshot.png)

---

## Features

- **Animated power flow diagram** — SVG arrows between Solar, Grid, Home and Battery nodes, animated in the direction of current flow with thickness proportional to power level.
- **Battery SOC ring** — coloured progress arc showing state of charge at a glance.
- **2×2 stat grid** — large power value, state badge (Generating / Charging / Importing…) and optional sparkline for each source.
- **Device row** — horizontal chip list showing individual device power consumption, sorted by wattage.
- **Auto dark/light theme** — follows the HA colour scheme or can be forced.
- **HACS-compatible** — single compiled JS file, no server required.

## Installation

### HACS (recommended)

1. Open HACS in your Home Assistant sidebar.
2. Go to **Frontend** → click the three-dot menu → **Custom repositories**.
3. Add `https://github.com/YOUR_GITHUB_USER/solar-overview-card` with category **Lovelace**.
4. Search for **Solar Overview Card** and click **Download**.
5. Reload your browser.

### Manual

1. Download `solar-overview-card.js` from the [latest release](https://github.com/YOUR_GITHUB_USER/solar-overview-card/releases/latest).
2. Copy it to `config/www/solar-overview-card.js`.
3. In Home Assistant go to **Settings → Dashboards → Resources** and add:
   ```
   URL:  /local/solar-overview-card.js
   Type: JavaScript module
   ```
4. Reload the browser.

## Configuration

Add a card to your Lovelace dashboard:

```yaml
type: custom:solar-overview-card

# ── Required entities ─────────────────────────────────────────────────────────
solar:
  entity: sensor.solar_power          # Watts generated by PV panels (≥ 0)
  name: Solar                         # Optional display name

battery:
  entity: sensor.battery_power        # Watts: positive = charging, negative = discharging
  soc_entity: sensor.battery_soc      # Optional: state of charge 0–100 %
  name: Battery

grid:
  entity: sensor.grid_power           # Watts: positive = importing, negative = exporting
  name: Grid

load:
  entity: sensor.load_power           # Watts consumed by the home (≥ 0)
  name: Home

# ── Optional ──────────────────────────────────────────────────────────────────
watt_threshold: 1000          # Display as kW above this value (default: 1000)
show_sparklines: true         # Show 2-hour history sparklines (default: true)
theme: auto                   # 'auto' | 'light' | 'dark'  (default: auto)

# ── Device list (optional) ────────────────────────────────────────────────────
devices:
  - entity: sensor.dishwasher_power
    name: Dishwasher
    icon: mdi:dishwasher        # MDI icon name (rendered inline as SVG path)
  - entity: sensor.ev_charger_power
    name: EV Charger
    icon: mdi:ev-station
  - entity: sensor.aircon_power
    name: Air Con
```

### Sign conventions

| Sensor | Positive | Negative |
|--------|----------|----------|
| `solar.entity` | Generating | — (always ≥ 0) |
| `battery.entity` | Charging | Discharging |
| `grid.entity` | Importing | Exporting |
| `load.entity` | Consuming | — (always ≥ 0) |

### Configuration reference

| Key | Type | Default | Description |
|-----|------|---------|-------------|
| `solar.entity` | string | **required** | Solar generation power sensor |
| `solar.name` | string | `"Solar"` | Display name |
| `solar.icon` | string | built-in | SVG path data for custom icon |
| `battery.entity` | string | **required** | Battery power sensor |
| `battery.soc_entity` | string | — | Battery state of charge sensor (0–100) |
| `battery.name` | string | `"Battery"` | Display name |
| `grid.entity` | string | **required** | Grid power sensor |
| `grid.name` | string | `"Grid"` | Display name |
| `load.entity` | string | **required** | Home consumption power sensor |
| `load.name` | string | `"Home"` | Display name |
| `devices` | list | `[]` | List of individual device sensors (see above) |
| `watt_threshold` | number | `1000` | W value above which kW is shown |
| `show_sparklines` | boolean | `true` | Render 2-hour history sparklines |
| `theme` | string | `"auto"` | `auto`, `light`, or `dark` |

## Building from source

```bash
git clone https://github.com/YOUR_GITHUB_USER/solar-overview-card.git
cd solar-overview-card
npm install
npm run build        # output: dist/solar-overview-card.js
npm run dev          # watch mode
```

Requires Node.js ≥ 18.
